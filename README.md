# Hello Recruto Service

Простой веб-сервис на **FastAPI**, который возвращает приветственное сообщение с параметрами из GET-запроса.

---

## Пример работы

Запрос:

```
http://localhost:2367/?name=Recruto&message=Давай дружить
```

Ответ:

```
Hello Recruto! Давай дружить!
```

---

## Установка зависимости

```bash
pip install fastapi uvicorn
```

---

## Запуск

```bash
uvicorn main:app --host 0.0.0.0 --port 2367
```

После запуска сервис будет доступен по адресу:

```
http://localhost:2367/
```

---

## Использование

* `name` — имя для приветствия
* `message` — сообщение

Пример GET-запроса:

```
http://localhost:2367/?name=Recruto&message=Давай дружить
```

Сервис возвращает текст с корректной поддержкой кириллицы.

---

## Примечания

* FastAPI автоматически валидирует параметры
* Ответ всегда в `UTF-8`
* Подходит для тестовых заданий и небольших проектов
